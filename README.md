# Dibujar Carta

Proyecto web realizado con HTML, CSS y JavaScript para generar y mostrar una carta visual de forma dinamica.

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## Objetivo del proyecto

Practicar la interaccion entre estructura, estilos y logica basica en JavaScript mediante la generacion dinamica de una carta en pantalla.

## Funcionalidades

- Renderizado visual de una carta
- Interaccion basica con JavaScript
- Estilos personalizados con CSS
- Estructura simple y clara para practica frontend

## Lo que practique

- Manipulacion basica del DOM
- Integracion de JavaScript con HTML y CSS
- Organizacion de un proyecto web simple
- Resolucion de pequenos problemas de logica visual
